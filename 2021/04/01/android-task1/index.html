android作业一

发表于2021-04-01|更新于2021-04-01|android
|阅读量:

1. 功能说明

完成一个类微信界面的应用，本次是4个tab切换效果的完成。

  1. 完成微信下面四个图标的展示
  2. 实现点击图标切换页面
  3. 实现图标状态切换颜色功能

2. 功能分析

2.1 底部图标的排列

  1. 创建一个横向的LinearLayout作为背景，用于排列图标

    <LinearLayout
        android:layout_width="match_parent"
        android:layout_height="80dp"
        android:orientation="horizontal"
        android:baselineAligned="false"
        >
</LinearLayout>

  2. 在其内部创建4个纵向的LinearLayout作为图标

    <LinearLayout
            android:id="@+id/id_tab_chat"
            android:layout_width="0dp"
            android:layout_height="match_parent"
            android:layout_weight="1"
            android:gravity="center"
            android:orientation="vertical"
            android:background="#F7F7F7"
            >
            <ImageView
                android:id="@+id/id_tab_chat_img"
                android:layout_width="match_parent"
                android:layout_height="wrap_content"
                app:srcCompat="@drawable/bottom_chat"
                android:clickable="false"
                android:contentDescription="@string/app_name"
                />
            <TextView
                android:id="@+id/id_tab_chat_text"
                android:layout_width="match_parent"
                android:layout_height="wrap_content"
                android:text="微信"
                android:gravity="center"
                android:clickable="false"
                android:textSize="15sp"
                />
</LinearLayout>

  3. 在activity_main中引入

    <include
        layout="@layout/bottom_chat"
        android:layout_width="match_parent"
        android:layout_height="wrap_content" />

2.2 实现点击切换页面

  1. 编写相关fragment

    <TextView
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        android:text="微信"
        android:textSize="30sp"
        android:gravity="center"
        android:textColor="@color/black"/>

  2. 在MainActivity.java中定义变量并初始化

    private Fragment chat = new Chat();
private Fragment find = new Find();
private Fragment mailList = new MailList();
private Fragment me = new Me();

private ImageView imgChat;
private ImageView imgMailList;
private ImageView imgFind;
private ImageView imgMe;

private LinearLayout llChat;
private LinearLayout llMailList;
private LinearLayout llFind;
private LinearLayout llMe;

private FragmentManager fm;


private void initFragment(){
    fm = getSupportFragmentManager();
    FragmentTransaction transaction = fm.beginTransaction();
    transaction.add(R.id.id_content, chat);
    transaction.add(R.id.id_content, find);
    transaction.add(R.id.id_content, mailList);
    transaction.add(R.id.id_content, me);
    transaction.commit();
}

private void initView(){
    llChat = findViewById(R.id.id_tab_chat);
    llMailList = findViewById(R.id.id_tab_mailList);
    llFind = findViewById(R.id.id_tab_find);
    llMe = findViewById(R.id.id_tab_me);

    imgChat = findViewById(R.id.id_tab_chat_img);
    imgMailList = findViewById(R.id.id_tab_mailList_img);
    imgFind = findViewById(R.id.id_tab_find_img);
    imgMe = findViewById(R.id.id_tab_me_img);
}

  3. 编写显示规则

    @Override
public void onClick(View v) {
    switch (v.getId()){
        case R.id.id_tab_chat:
            selectFragment(0);
            break;
        case R.id.id_tab_mailList:
            selectFragment(1);
            break;
        case R.id.id_tab_find:
            selectFragment(2);
            break;
        case R.id.id_tab_me:
            selectFragment(3);
            break;
    }
}

private void selectFragment(int i){
    FragmentTransaction transaction = fm.beginTransaction();
    // 隐藏Fragment
    hideFragment(transaction);
    switch (i){
        case 0:
            transaction.show(chat);
            break;
        case 1:
            transaction.show(mailList);
            break;
        case 2:
            transaction.show(find);
            break;
        case 3:
            transaction.show(me);
            break;
        default:
            break;
    }
    transaction.commit();
}

private void hideFragment(FragmentTransaction transaction){
    transaction.hide(chat);
    transaction.hide(mailList);
    transaction.hide(find);
    transaction.hide(me);
}

2.3 实现颜色切换

  1. 新建一个绿色图标，只需修改原图标颜色为绿色

  2. 设置点击时重置所有图标颜色，并将所点击的图标切换为绿色图标

    private void resetImg(){
    imgChat.setImageResource(R.drawable.bottom_chat);
    imgMailList.setImageResource(R.drawable.bottom_mail_list);
    imgFind.setImageResource(R.drawable.bottom_find);
    imgMe.setImageResource(R.drawable.bottom_me);
}

private void selectFragment(int i){
    FragmentTransaction transaction = fm.beginTransaction();
    // 隐藏Fragment
    hideFragment(transaction);
    switch (i){
        case 0:
            transaction.show(chat);
            // 设置点击时显示状态(图标)
            imgChat.setImageResource(R.drawable.green_bottom_chat);
            break;
        case 1:
            transaction.show(mailList);
          imgMailList.setImageResource(R.drawable.green_bottom_mail_list);
            break;
        case 2:
            transaction.show(find);
            imgFind.setImageResource(R.drawable.green_bottom_find);
            break;
        case 3:
            transaction.show(me);
            imgMe.setImageResource(R.drawable.green_bottom_me);
            break;
        default:
            break;
    }
    transaction.commit();
}
